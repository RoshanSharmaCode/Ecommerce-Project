# 🛒 E-commerce Project

A full-featured e-commerce web application built with React, demonstrating real-world development concepts like routing, backend integration, state management, and testing.

---

## 🚀 Features

* Multi-page navigation using React Router
* Fetching data from backend APIs
* Add to cart functionality (data mutation)
* Dynamic product listing
* Automated testing using Vitest

---

## 🛠️ Tech Stack

* **Frontend:** React.js, JavaScript, CSS
* **Routing:** React Router
* **Backend:** Node.js / Express
* **Testing:** Vitest

---

## ▶️ How to Run

### 1. Start Backend

* Go to `ecommerce-backend` folder
* Open in VS Code
* Open Terminal (`Terminal > New Terminal`)
* Run:

  ```bash
  npm install
  npm run dev
  ```

### 2. Start Frontend

* Open a new terminal
* Go to `ecommerce-project` folder
* Run:

  ```bash
  npm install
  npm run dev
  ```

### 3. Open App

* Open the URL shown in the terminal in your browser

---

## 📂 Project Structure

```
ecommerce-backend/   → Backend API  
ecommerce-project/   → Frontend React App  
```

---

## 🧪 Testing

Run tests using:

```bash
npm run test
```

---

## 📌 Note

If the app doesn’t load properly ensure both frontend and backend are running.
